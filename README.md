Lista de Compras (CRUD Console App) - Versão 1.0
Este é o meu primeiro projeto de CRUD (Create, Read, Update, Delete) desenvolvido em C# utilizando o .NET.
É uma aplicação simples de console voltada para o gerenciamento de uma lista de compras, servindo como base inicial para o meu portfólio.

O objetivo deste projeto foi praticar conceitos fundamentais da linguagem C#, como:
- Estruturas de repetição (while)
- Estruturas de decisão (switch)
- Listas (List<T>)
- Entrada e saída de dados (Console.ReadLine e Console.WriteLine)
- Manipulação de coleções
- Organização da lógica de um sistema simples
O sistema simula uma lista de compras onde o usuário pode adicionar, remover, visualizar e limpar itens diretamente pelo terminal.

Tecnologias Utilizadas
